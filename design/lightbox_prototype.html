DESIGN PROTOTYPE — STATIC ONLY
This file demonstrates a lightweight, dependency-free image lightbox intended for future use inside AudioAtlas report.html plot cards. It contains no Python, no production logic, and does not modify any reports or analysis. Open this file directly in any browser.

AudioAtlas Lightbox Prototype

Calm, offline, keyboard-friendly image viewer for plot cards. Click any plot image below.
Try it: Click a thumbnail → larger overlay opens. Use ← → arrow keys, Prev/Next buttons, or click the dark backdrop to close. Press Esc to close. All images are self-contained vector placeholders (no network).

Plots

In a real report these would be the 10 generated PNGs. The lightbox lets you inspect any of them at larger size while staying in the page.

Waveform + RMS Envelope

01_waveform_rms.png

Raw samples with overlaid RMS envelope (track-normalized).

Frame RMS Timeline

02_rms_timeline.png

Frame-by-frame RMS (dBFS) showing dynamic contour over time.

Sample Histogram

05_sample_histogram.png

Distribution of sample values with clipping and near-clipping thresholds marked.

Stereo Correlation Timeline

06_stereo_correlation.png

Per-frame Pearson correlation between left and right channels.

Mid/Side Energy Timeline

07_mid_side_energy.png

Mid and side RMS over time plus side-to-mid ratio panel.

Spectral Shape Timeline

08_spectral_shape.png

Time-varying spectral centroid, 85%/95% rolloff, and bandwidth.

Log-Frequency Spectrogram

03_log_spectrogram.png

STFT magnitude, log frequency axis, normalized to track maximum (0 dB). Not calibrated dBFS.

Welch Average Spectrum

04_average_spectrum.png

Average power spectrum (Welch) with labeled frequency anchors. Relative to track maximum.

Frequency Band Energy Timeline

09_band_energy_timeline.png

Relative energy in sub / bass / low_mid / mid / presence / high / air bands (track-normalized).

Onset / Transient Density Timeline

10_onset_density.png

Smoothed onset strength (transient activity) with relative high-density regions highlighted.

This prototype is purely exploratory. Real report.html would continue to render the exact same plot cards (article.plot-card + img + .plot-filename + .plot-desc). The lightbox script below is self-contained and can be inlined into a future template with only minor adaptation (query the existing DOM nodes).

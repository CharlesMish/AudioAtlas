# Security policy

## Supported versions

AudioAtlas is a public alpha. Security fixes are applied to the newest published
alpha only. Upgrade to the latest release before reporting behavior that is
already fixed there.

## Reporting a vulnerability

Use GitHub's private vulnerability-reporting form for this repository:

https://github.com/CharlesMish/AudioAtlas/security/advisories/new

Include the AudioAtlas version, operating system, Python version, affected
command or artifact, and the smallest safe reproduction you can provide. Do not
attach proprietary, unreleased, or personally identifying audio. A synthetic or
rights-cleared fixture is preferred.

Please do not open a public issue for a vulnerability until a fix or coordinated
disclosure plan is available.
